
node-multi-hashing
===============

> from https://github.com/cryptorex/node-multi-hashing


Cryptocurrency hashing functions for node.js.

----------
* quark
* x11
* x13
* nist5
* scrypt
* scryptn
 
* keccak
* bcrypt
* skein
* groestl
* blake
* fugue
* qubit
* hefty1
* shavite3
* cryptonight
* boolberry
* sha256d
* lbry
